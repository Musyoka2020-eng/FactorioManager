Replace widgets.py completely. The file has two classes: `Notification` and `NotificationManager`.

**Critical implementation notes (from RESEARCH.md):**
1. `QFrame.setWindowOpacity()` silently does NOTHING for embedded child widgets — MUST use `QGraphicsOpacityEffect`.
2. After calling `setProperty("notifType", value)`, call `self.style().unpolish(self); self.style().polish(self)` so QSS re-evaluates the dynamic property.
3. Store `QPropertyAnimation` on `self._anim` to prevent garbage collection before it finishes.
4. `NotificationManager` must call `raise_()` on each new notification so it renders above tab content.

**Notification class:**
- Parent: `QFrame`, child of `container` (passed by NotificationManager)
- `setFixedWidth(420)` — max width from UI-SPEC.md
- Dynamic property `notifType` applied on init; QSS handles all coloring via `QFrame[notifType=...]`
- Layout: `QHBoxLayout` → icon `QLabel` (14pt Bold, right margin 10px) + message `QLabel` (10pt Normal, setWordWrap(True)) + optional × close button
- For persistent toasts: add bottom `QHBoxLayout` with action buttons
- Auto-dismiss: `QTimer.singleShot(duration_ms, self._start_fade)` if duration_ms > 0
- Fade: `QGraphicsOpacityEffect` + `QPropertyAnimation(effect, b"opacity")`, duration 300ms, 1.0→0.0, `finished.connect(self.deleteLater)`
- `dismissed` signal (no args) emitted when deleted — NotificationManager uses this to remove from active list

**NotificationManager class:**
- `__init__(self, container: QWidget)`: receives `centralWidget()`
- `self._active: list[Notification]` = []
- `show(message, notif_type="info", duration_ms=4000, actions=None)`: 
  1. Creates `Notification(self._container, message, ...)` 
  2. Calls `notif.adjustSize()` then `.raise_()`
  3. Calls `self.reposition_all()` to snap position
  4. Shows with `notif.show()`
  5. Connects `notif.dismissed` to `lambda: self._on_dismissed(notif)`
  6. Appends to `self._active`
- `reposition_all(self)`: calculates position of each active notification. Right edge: `container.width() - 16`. Top start: 16px from top of container. Stack downward with 8px gap. Call `notif.move(x, y)` for each visible notification.
- `_on_dismissed(self, notif)`: removes from `self._active`

Write the complete implementation including all imports from PySide6.

Close button (×): always shown on persistent toasts; shown on auto-dismiss only if explicitly requested. Positioned in the top-right corner of the notification QFrame.
