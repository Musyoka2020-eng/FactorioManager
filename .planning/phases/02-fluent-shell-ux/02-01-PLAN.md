## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Portal API → Notification message | Mod names/descriptions from Factorio portal API can contain HTML-like content that reaches QLabel in notifications |
| User input → QLabel display | Mod name/author fields fetched via portal and displayed in QLabel widgets |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-02-01 | Spoofing / Tampering | `Notification._build_ui()` QLabel | mitigate | Set `msg_label.setTextFormat(Qt.TextFormat.PlainText)` to prevent portal API HTML content from rendering as rich text in toast notifications (Task 2, step F) |
| T-02-02 | DoS | `NotificationManager._keyed` dict | accept | Dict size bounded by `_MAX_ACTIVE` cap already in place; event keys are caller-controlled strings, not user input from untrusted sources |
| T-02-03 | Information Disclosure | QSS token values in dark_theme.qss | accept | QSS is a local resource file; no credentials or secrets — purely visual constants |
</threat_model>
